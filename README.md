# JS Calculator Project

Простой калькулятор на JavaScript с поддержкой базовых арифметических операций (`+`, `-`, `*`, `/`).

## Функции

- Кнопки для цифр (0–9) и операций (+, -, *, /)
- Кнопка `C` для очистки всего ввода
- Кнопка `=` для вычисления выражения
- Кнопка `←` для удаления последнего символа
- Поддержка отрицательных чисел
- Простая и безопасная функция вычислений с проверкой ввода

## Установка и запуск

1. Клонируем репозиторий:

```bash
git clone <URL_REPOSITORY>
cd cicd
```
2. Установить зависимости:

```bash
npm install
```
3. Запуск:

- Запуск через index.html

## CI/CD
